# AdventureWorks
Bu repository AdventureWorks verilənlər bazası əsasında hazırlanmış Power BI dashboard-larını ehtiva edir. Analizlər Satış, Alış, Müştəri və Məhsul istiqamətindədir.

📊 Dashboard-lar

##Satış: Satış performansı sifariş sayı və satış həcmi əsasında təhlil edilir. Tarix və territoriya üzrə satış trendləri, sifariş ölçüsünə görə paylanma və satış məbləği ilə satış miqdarı arasındakı əlaqə araşdırılır.

-Alış: Alış fəaliyyəti ümumi alış məbləği, sifariş sayı və məhsul miqdarı əsasında təhlil edilir. Tarix üzrə trendlər, əsas müştərilər və məhsullar üzrə alış, geri qaytarılma və stok vəziyyəti analiz olunur. 

-Müştəri seqmentasiyası, əsas müştərilərin satış göstəriciləri, territoriyalar üzrə paylanma və zaman üzrə müştəri sayının dinamikası təhlil olunur.

-Məhsul analizi çərçivəsində satış üzrə ən yüksək göstəriciyə malik məhsullar müəyyən edilir, məhsul miqdarı ilə satış məbləği arasındakı əlaqə təhlil olunur və kateqoriya və subkateqoriyalar üzrə satış strukturu qiymətləndirilir.
